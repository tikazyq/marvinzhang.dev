The 2021 Golang piece scored every celebrated feature against its shadow. Run Bun through the same table.

The other edge

Speed-first engineering culture
Benchmarks and iteration pace pulled 21% of surveyed developers (State of JS 2025, +4pt YoY)
Long-running-process memory leaks; OpenCode migrated off citing “memory issues, crashes, terrible Windows support”
Fast-moving, AI-accelerated development
Rapid feature delivery; 1.x line shipped runtime, SQL, Redis, full-stack dev server in two years
Stability discount in production; the canary-vs-stable paradox — Prisma trusted the build that passed its tests, not the version label
Community consensus is itself a verdict: bun install for development speed, Node in production — loop-latency optimization cannot substitute for runtime reliability.
