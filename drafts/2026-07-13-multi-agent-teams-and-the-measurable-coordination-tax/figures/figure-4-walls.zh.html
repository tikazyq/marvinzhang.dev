图 04 — 可测量性

组织经济学的三堵墙，agent 一次拆完

让“协调成本”测不了的三个百年难题，与 agent 系统给出的解法。

墙一：没有日志

走廊里的闲聊、会议室里的眼神，都不留记录——公司没法打 log

→

每次交互都走 API

token 用量、时间戳、完整内容，天然全量记账

墙二：不能重跑

历史只发生一次；没有公司能用 8 人和 16 人各过一遍同一个季度

→

N 是自由变量

同一批任务，2 / 4 / 8 / 16 个 agent 各跑一遍，其他全部锁死

墙三：人会应付指标

古德哈特定律：指标一旦用于考核，就不再是好指标

→

agent 不应付账本

它不知道自己被测量，也不为绩效虚报工作量

测量的两条路线（协调成本 ≠ 消息 token 数）

结果侧 · 整体测量

固定任务集、扫描 N，看曲线弯离理想直线多少——弯掉的就是协调税，无需拆解归因（Silo-Bench 的 RCC 走的就是这条路）

过程侧 · 指标篮子

上下文重复率、agent 利用率、每次委派成本、错误放大倍数——每个都只是一个侧面，合起来当仪表盘看

对人类团队，那些数字只能算；对 agent 团队，第一次可以量——人月神话从格言变成可复现实验的定律。
